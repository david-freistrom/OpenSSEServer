# OpenSSEServer

Server-Sent-Events SSE Server Implementierung 

## Install

```
$ npm install --global open-sse-server
```

## Usage

sse-server [-c settings.json]

## Developing

```
OpenIPT$ npm install
OpenIPT$ grunt build
OpenIPT$ grunt test
